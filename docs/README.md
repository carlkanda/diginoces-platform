# Diginoces Documentation Index

This folder contains the version-controlled documentation snapshots used to guide the AI-agent build process for the Diginoces platform.

## Folder structure

```text
docs/
  product/
  agent-system/
  backlog/
  technical-design/
  planning/
```

## Source of truth policy

Google Drive remains the editable business/workshop source. GitHub contains version-controlled snapshots used by developers and AI agents.

## Required build rule

No feature may be implemented unless it is linked to a documented requirement ID, and no requirement may be marked complete unless it has been implemented, tested, and reviewed.
